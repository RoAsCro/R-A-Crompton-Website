Un Passant

Created for the final year project of my Master's, Un Passant is a Java program for solving last n moves retrograde analysis chess problems, such as those found in Raymond Smullyan's The Chess Mysteries of Sherlock Holmes. The program can take a chess position and use layers of deductive reasoning to analyse the position and determine a. whether that position is reachable in a conventional game of chess, and b. what, if any, the last n number of moves were.

The program uses a mix of depth-first searches with backtracking together with a series of complex heuristics to determine a board state's legality.
